# Visual Examples

This directory contains visual examples of Unreal Miner outputs.

## Gallery Images

### Required Screenshots

To complete the gallery section, add the following images:

1. **satellite_input.png** - Sentinel-1 SAR data visualization
2. **heightmap_output.png** - Processed heightmap output
3. **unreal_visualization.png** - Interactive 3D terrain in Unreal Engine
4. **anomaly_map.png** - AI-detected geological anomalies overlay
5. **workflow_diagram.png** - Pipeline workflow visualization (optional)

### Image Guidelines

- **Format**: PNG (preferred) or JPEG
- **Size**: Optimized for web viewing (< 1MB per image)
- **Resolution**: 1920x1080 or 16:9 aspect ratio recommended
- **Content**: Clear, annotated screenshots showing real outputs

### Adding Images

1. Process sample data or your own tile
2. Take screenshots of each pipeline stage
3. Add descriptive captions in README.md
